# MiscSpatialData

## Overview

Miscellaneous spatial data sets created by the United States Geological Survey (USGS)
Idaho National Laboratory (INL)
[Project Office](https://www.usgs.gov/centers/id-water/science/idaho-national-laboratory-project-office "USGS INL Project Office").
